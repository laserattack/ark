# Usage

```
chains - simple script for making incremental backups
Usage: chains [-hifd]

Flags:
  -h, --help         This help
  -i, --incremental  Make incremental backup
  -f, --full         Make full backup
  -d, --dir DIR      Backup dir (default: /home/serr/.chains)

Notes:
  1. TAR IS REQUIRED
```
